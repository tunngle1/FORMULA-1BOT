# F1 Friend League Bot (Demo)

Демо-бот для дружеских ставок на квалификацию F1 с использованием OpenF1 API.

## Функционал

- Берёт последний прошедший Гран-при выбранного сезона.
- Загружает данные квалификации из OpenF1.
- Открывает приём ставок на Q1 (по трёхбуквенным кодам пилотов: VER, NOR, HAM и т.д.).
- Закрывает приём ставок и считает очки:
  - +2 очка за каждого пилота, который реально вылетел в Q1 (P16–P20).

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

1. Создай бота через @BotFather, получи токен.
2. Экспортируй токен в переменную окружения:

```bash
export TELEGRAM_TOKEN=123456:ABC...
python bot.py
```

## Использование в Telegram

В чате с ботом:

```text
/start
/setyear 2024          # или другой доступный год
/demo_init             # бот найдёт последний ГП этого сезона и подготовит данные
/q1_open               # открыть приём ставок на Q1
/bet_q1 ver nor ham pia alo
/q1_close              # показать реальный Q1 OUT и очки по Q1
```

Дальше бота можно расширять под Q2/Q3, гонку и полноценную лигу.
